使用说明

  • 容器会初始化，产生一个全局变量SmkJSBridge, 然后触发此事件，SmkJSBridge注入是一个异步过程，因此尽可能先监听该事件后再call接口。 代码演示：
function ready(callback) {
  if (window.SmkJSBridge) {
  callback && callback();
  } else {
  document.addEventListener('SmkJSBridgeReady', callback, false);
  }
}
